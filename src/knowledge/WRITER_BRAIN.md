# РАЗДЕЛ 1: ЖЕЛЕЗНЫЕ ПРАВИЛА WRITER

## 1.1 ВСЕГДА (MUST DO)

### Структура текста

- **W-RULE-001**: Первые 125 символов — это ВСЁ. Instagram обрезает текст после ~125 символов (2 строки) и показывает «...ещё». Hook ОБЯЗАН быть в этих 125 символах. Если человек не нажмёт «ещё» — остальной текст не существует.

- **W-RULE-002**: Каждый абзац — 1-3 предложения. Стена текста = смерть engagement. Между абзацами — ПУСТАЯ СТРОКА. Это не проза, это mobile-first контент.

- **W-RULE-003**: Чередуй ритм: короткое предложение → среднее → короткое. «Четыре недели. Ни одного скола. Вот что значит правильная база и техника нанесения в три слоя.» Монотонный ритм усыпляет.

- **W-RULE-004**: CTA — ВСЕГДА последний абзац. Не вставляй CTA в середину текста. Читатель должен пройти путь: hook → интерес → желание → действие.

- **W-RULE-005**: Эмодзи — 2-5 на пост. Используй как визуальные разделители и акценты, НЕ как замену слов. Основные: 💅 ✨ 🤍 📩 👇. НЕ использовать: 🔥🔥🔥 (спам), ❤️❤️❤️ (спам), 💯 (дёшево выглядит).

- **W-RULE-006**: Длина текста зависит от типа поста:
  - Portfolio: 100–300 символов (лаконично)
  - Educational: 500–1200 символов (ценность)
  - Testimonial: 400–800 символов (история)
  - Behind the Scenes: 200–500 символов (лёгкость)
  - Promo: 150–400 символов (чёткость)
  - Personal Story: 500–1000 символов (теплота)

### Язык и стиль

- **W-RULE-007**: Пиши как ГОВОРИШЬ. Не как пишут в учебнике. Тест: прочитай вслух. Если звучит неестественно — перепиши.

- **W-RULE-008**: Конкретика > абстракции. НЕ «долго держится» → «до 4 недель без сколов». НЕ «красивый дизайн» → «нюдовый градиент с тонкой линией у кутикулы». НЕ «качественные материалы» → «база Kodi + топ Neonail».

- **W-RULE-009**: Одна мысль = один пост. Не пытайся рассказать всё. Пост про стойкость — только про стойкость. Пост про кутикулу — только про кутикулу.

- **W-RULE-010**: Обращение к читателю — на «вы» в feed-постах (русский), на «את» в иврите. В Stories допускается «ты» / «את». Ивритский текст может быть менее формальным чем русский — это норма для израильских соцсетей.

- **W-RULE-011**: Tone of Voice = «тёплый профессионал». Представь женщину-эксперта 30-40 лет, которой доверяешь. Не подруга («Девочки, ну вы не поверите!!!»), не корпорация («Наша компания рада сообщить...»), не блогер-подросток («Ваааау, это просто бомба!!!»).

- **W-RULE-012**: Каждый пост на ОДНОМ языке. Русский пост = полностью русский (кроме терминов: гель-лак, nail art). Ивритский = полностью иврит. НИКОГДА не смешивать.

### Уникальность

- **W-RULE-013**: Каждый hook уникален. НЕ повторять hook чаще чем 1 раз в 30 дней. Перед написанием — проверить список предыдущих hooks из brief'а.

- **W-RULE-014**: USP мастера НЕ повторять дословно. «Стойкое покрытие до 4 недель + аккуратная кутикула без травм» — максимум 1 раз в 10 постов дословно. В остальных — раскрывать через истории, примеры, сравнения, процессы.

- **W-RULE-015**: Для каждого brief'а — минимум 3 варианта hook перед выбором финального.

## 1.2 НИКОГДА (MUST NOT DO)

### Запрещённые слова и фразы

- **W-BAN-001**: СТОП-СЛОВА (русский):
  - «качественный» → заменять на конкретику («трёхслойное нанесение», «база с каучуком»)
  - «профессиональный» → показывать через действие, а не декларировать
  - «лучший» → субъективно и пусто, заменять на факт или результат
  - «уникальный» → клише, не несёт смысла
  - «идеальный» → слишком часто используется, заменять на конкретное описание
  - «волшебный/магический» → инфантильно для ЦА 35-45
  - «гарантируем» → юридически опасно, не обещать невозможного
  - «недорого/дёшево» → обесценивает услугу
  - «акция! скидка! распродажа!» (с восклицательными знаками) → выглядит как спам

- **W-BAN-002**: СТОП-СЛОВА (иврит):
  - «!מבצע מטורף» (безумная акция) → спам
  - «הכי טוב» (самый лучший) → пусто
  - «מקצועי» (профессиональный) → показывай, не декларируй
  - «!רק היום» (только сегодня) → фальшивая urgency разрушает доверие

- **W-BAN-003**: НЕ НАЧИНАТЬ пост с:
  - Имени бренда/мастера («Marina Nails представляет...»)
  - Приветствия («Привет, девочки!», «Добрый день!»)
  - Вопроса «Как дела?» или подобной пустоты
  - Хештега
  - Эмодзи без текста

- **W-BAN-004**: НЕ ИСПОЛЬЗОВАТЬ клише beauty-текстов:
  - «Ваши ноготки заслуживают лучшего» → пусто
  - «Побалуйте себя» → избито
  - «Красота требует жертв» → токсично
  - «Время позаботиться о себе» → слишком общее
  - «Ногти — визитная карточка женщины» → клише
  - «В наших нежных руках» → слащаво

- **W-BAN-005**: НЕ выдумывать имена клиенток, конкретные цитаты, истории. Если нет реального отзыва — использовать обобщённый формат: «Обычно после первого визита клиентки говорят...» или «Вот что я слышу чаще всего после 3 недель носки...»

- **W-BAN-006**: НЕ давать медицинских обещаний. НЕ «вылечим грибок», НЕ «восстановим ногтевую пластину», НЕ «укрепим слабые ногти навсегда».

- **W-BAN-007**: НЕ унижать конкурентов. НЕ «в отличие от других мастеров», НЕ «не как в дешёвых салонах», НЕ «наконец-то нормальный мастер».

- **W-BAN-008**: НЕ использовать больше 3 восклицательных знаков на весь пост. Восклицательные знаки — как чеснок: чуть-чуть усиливает, много — убивает.

# РАЗДЕЛ 2: ШАБЛОНЫ ПО ТИПАМ ПОСТОВ

## 2.1 PORTFOLIO (описание работы)

### Структура:
```
СТРОКА 1: Hook — остановить скролл (что особенного в этой работе?)
СТРОКА 2-3: Описание работы — техника, цвет, детали
СТРОКА 4: Социальное доказательство ИЛИ микро-история
СТРОКА 5: CTA
```

### Длина: 100–300 символов
### Тон: Лаконичный, уверенный, визуальный

### Примеры (русский):

**Пример 1 — акцент на стойкости:**
```
Через 3 недели после нанесения. Ни одного скола 💅

Нюдовый камуфляж + выравнивание архитектуры.
Обработка кутикулы аппаратом — без порезов, без покраснений.

Клиентка приходит раз в 4 недели и каждый раз удивляется.

📩 Запись → WhatsApp в шапке профиля
```

**Пример 2 — акцент на дизайне:**
```
Молочный с тонкими паутинками ✨

Когда хочется что-то нежное, но не банальное.
Эффект невесомости на длинном миндале.

Носится уже 18 дней — всё на месте.

Хотите подобрать свой вариант? → Напишите в WhatsApp 📩
```

**Пример 3 — акцент на трансформации:**
```
Листай 👉 Было → Стало

Ногти после самостоятельного снятия: тонкие, с бороздками.
Два визита: мягкое снятие + восстанавливающая база + правильная архитектура.

Результат видите сами.

📩 Запись через WhatsApp → ссылка в bio
```

**Пример 4 — акцент на цвете:**
```
Терракотовый на коротком квадрате 🤎

Этот оттенок — как латте с корицей.
Сдержанный, тёплый, ко всему подходит.

Цвет держится без потери яркости все 4 недели.

Хотите такой же? → Пишите в WhatsApp 📩
```

**Пример 5 — минимализм:**
```
Когда покрытие нанесено правильно — видно по бликам ✨

Ровная поверхность. Чистая кутикула. Никакого затёка.

Детали, которые отличают аккуратную работу от «и так сойдёт».

📩 Запись открыта → WhatsApp в bio
```

### Примеры (иврит):

**Пример 1:**
```
3 שבועות אחרי — אפס שברים 💅

קמופלאז׳ ניוד + ארכיטקטורה מושלמת.
טיפול עדין בקוטיקולה — בלי חתכים, בלי אודם.

רוצה גם? → שלחי הודעה בווטסאפ 📩
```

**Пример 2:**
```
חלבי עם חוטי עכביש ✨

כשרוצים משהו עדין אבל לא בנאלי.
אפקט של קלילות על שקד ארוך.

📩 לקביעת תור → ווטסאפ בביו
```

**Пример 3:**
```
תחליקי ימינה 👉 לפני → אחרי

ציפורניים אחרי הסרה עצמית: דקות, עם חריצים.

התוצאה מדברת בעד עצמה.
```

### DO:
- Описывай КОНКРЕТНО: оттенок, форму ногтя, технику
- Упоминай срок носки если он впечатляет
- Используй визуальные метафоры для цвета

### DON'T:
- Не пиши «красивый маникюр» — это пусто, фото само показывает
- Не перечисляй все услуги — один пост = одна работа
- Не ставь цену в каждом portfolio-посте (только если это в brief'е)

## 2.2 EDUCATIONAL (советы, мифы, FAQ)

### Структура:
```
СТРОКА 1: Hook — вопрос или мини-провокация
СТРОКА 2-3: Раскрытие проблемы / мифа
СТРОКА 4-6: Объяснение / совет (разбито на пункты БЕЗ bullet-points, через абзацы)
СТРОКА 7: Резюме-вывод
СТРОКА 8: CTA (мягкий: «сохрани» / «отправь подруге»)
```

### Длина: 500–1200 символов
### Тон: Экспертный, дружелюбный, без менторства

### Примеры (русский):

**Пример 1 — миф:**
```
«Ногтям нужно отдыхать от гель-лака» — правда или миф? 🤔

Этот совет кочует из статьи в статью.
Но если разобраться — ногтевая пластина состоит из мёртвых клеток кератина.
Она не «дышит» и не «устаёт».

Ногти портятся не от покрытия, а от неправильного снятия.
Когда мастер спиливает слишком грубо или клиентка сдирает гель-лак сама — вот тогда пластина истончается.

Правильное нанесение + аккуратное снятие аппаратом = можно носить покрытие постоянно без вреда.

Сохрани 📌 чтобы показать подруге, которая до сих пор «даёт ногтям отдых»
```

**Пример 2 — совет:**
```
Почему маникюр скалывается через неделю?
Три самых частых причины 👇

Первая: руки мокли в горячей воде в первые сутки после нанесения. Гель-лак набирает прочность постепенно — первые 24 часа лучше избегать длительного контакта с горячей водой.

Вторая: сухая кутикула. Когда кожа вокруг ногтя пересыхает — она начинает «тянуть» покрытие, и по краю появляются отслойки. Масло для кутикулы — ваш друг.

Третья: покрытие нанесено без подготовки. Если не убрать птеригий и не обезжирить пластину — сцепка будет слабой, и покрытие начнёт отслаиваться.

Какая из причин была у вас? Напишите в комментариях 👇
```

**Пример 3 — FAQ:**
```
Вопрос, который мне задают чаще всего:
«Аппаратный маникюр — это больно?» 😬

Отвечаю: нет, если мастер работает правильно.

Аппаратные фрезы вращаются, а не режут.
Они мягко убирают ороговевшую кожу, не затрагивая живую ткань.

Ощущение — лёгкая вибрация и тепло. Не боль.

Если вам когда-то было больно при аппаратном маникюре — проблема была не в технологии, а в технике конкретного мастера.

У меня клиентки засыпают 😴

Есть ещё вопросы? Задавайте в комментариях — отвечу на всё 👇
```

### Примеры (иврит):

**Пример 1:**
```
״הציפורניים צריכות לנוח מג׳ל״ — אמת או מיתוס? 🤔

המיתוס הזה עובר ממאמר למאמר.
אבל ציפורן עשויה מתאי קרטין מתים — היא לא ״נושמת״ ולא ״עייפה״.

מה שפוגע בציפורניים זו הסרה לא נכונה.
כשמסירים בכוח או שופכים יותר מדי — הציפורן נהיית דקה.

הסרה עדינה + בסיס נכון = אפשר לשים ג׳ל כל הזמן בלי נזק.

שמרי 📌 ושלחי לחברה שעדיין ״נותנת לציפורניים לנוח״
```

**Пример 2:**
```
למה הג׳ל מתקלף אחרי שבוע?
שלוש סיבות הכי נפוצות 👇

ראשית — ידיים רטובות במים חמים ב-24 השעות הראשונות. הג׳ל מתחזק בהדרגה — כדאי להימנע ממים חמים ביום הראשון.

שנית — קוטיקולה יבשה. כשהעור סביב הציפורן מתייבש, הוא ״מושך״ את הציפוי ויוצר הפרדות בקצוות.

שלישית — הכנה לא מספקת. בלי להסיר פטריגיום ולנקות שומנים — ההיאחזות חלשה.

איזו סיבה מוכרת לך? כתבי בתגובות 👇
```

### DO:
- Давай конкретную пользу, которую можно применить
- Разрушай мифы с уважением к читателю
- Заканчивай вопросом для engagement

### DON'T:
- Не используй менторский тон («Вы должны понимать...»)
- Не перегружай терминами без объяснения
- Не делай пост-лекцию длиннее 1200 символов

## 2.3 TESTIMONIAL (отзыв/история клиентки)

### Структура:
```
СТРОКА 1: Hook — цитата или ситуация
СТРОКА 2-3: Контекст (что было до / какая задача)
СТРОКА 4-5: Что сделали / процесс
СТРОКА 6: Результат
СТРОКА 7: CTA
```

### Длина: 400–800 символов
### Тон: Тёплый, искренний, с деталями

### ВАЖНО: Если отзыв реальный — использовать как есть (с разрешения). Если нет — ОБОБЩЁННЫЙ формат.

### Примеры (русский):

**Пример 1 — обобщённый формат (нет реального отзыва):**
```
«Я уже отчаялась найти мастера, у которого покрытие держится дольше двух недель»

Это я слышу на первом визите чаще всего.
И каждый раз приглашаю вернуться через три-четыре недели — просто чтобы увидели разницу.

Возвращаются. Удивляются. Остаются.

Дело не в «волшебном лаке» — дело в технике: правильная подготовка пластины, трёхслойное нанесение, запечатывание торца.

Хотите проверить на своих ногтях? 📩 WhatsApp → в шапке профиля
```

**Пример 2 — скриншот отзыва:**
```
Вот такие сообщения — лучшая мотивация 🤍

Скриншот выше — реальная переписка (с разрешения клиентки).

Три недели носки. Ни одного скола. И это при активном образе жизни: спортзал, готовка, уборка.

Когда покрытие нанесено правильно — оно не боится вашей жизни.

📩 Запись → WhatsApp в bio
```

**Пример 3 — история трансформации:**
```
Вот что бывает, когда находишь «своего» мастера.

Эта клиентка пришла ко мне полгода назад.
Ногти после наращивания в другом месте: тонкие, с бороздками, ломкие.

Мы не стали сразу наращивать.
Сначала — восстановление: укрепляющая база, мягкое снятие, уход.
Через три визита — ногти окрепли.
Сейчас — стабильная длина и здоровая пластина.

Терпение + правильный уход = результат.

📩 Хотите начать восстановление? → WhatsApp в bio
```

### Примеры (иврит):

**Пример 1:**
```
״כבר ויתרתי למצוא מישהי שהג׳ל מחזיק יותר משבועיים״

את המשפט הזה אני שומעת בביקור הראשון הכי הרבה.
ובכל פעם אני מזמינה לחזור אחרי 3-4 שבועות — רק כדי לראות את ההבדל.

חוזרות. מופתעות. נשארות.

זה לא ״לק קסום״ — זו טכניקה: הכנת הציפורן, שלוש שכבות, חיתום קצוות.

רוצה לבדוק בעצמך? 📩 ווטסאפ → בביו
```

**Пример 2:**
```
ההודעות האלה — המוטיבציה הכי טובה 🤍

צילום מסך אמיתי (באישור הלקוחה).
שלושה שבועות — אפס שברים. עם חדר כושר, בישולים, וניקיונות.
```

# РАЗДЕЛ 5: POWER WORDS

## 5.1 Русский

### Слова-усилители для описания результата:
- **Стойкость**: безупречный, стойкий, надёжный, без единого скола
- **Аккуратность**: чистый, точный, ювелирный, без затёков, без покраснений
- **Красота**: нежный, элегантный, сдержанный, тёплый, глубокий
- **Уникальность**: индивидуальный, под ваш стиль, подобранный лично
- **Комфорт**: мягкий, безболезненный, расслабляющий, деликатный

### Слова-триггеры (вызывают действие):
- осталось (2 окна, 3 места)
- уже / всё ещё (уже через 20 минут, всё ещё на месте через 3 недели)
- первая / первые (первая примерка нового оттенка)
- без (без боли, без сколов, без компромиссов)
- вот так / вот что (вот так выглядит, вот что бывает)

### Сенсорные слова (описывают визуал словами):
- **Текстура**: шелковистый, бархатный, глянцевый, матовый, зеркальный
- **Цвет через метафору**: молочный, карамельный, пыльная роза, вино, латте, терракот
- **Свет**: блик, свечение, мерцание, перелив, голограмма

## 5.2 Иврит (עברית)

### Слова-усилители:
- **עמידות** (стойкость): עמיד, חזק, ללא שברים, מחזיק שבועות
- **דיוק** (точность): עדין, מדויק, נקי, ללא פגיעה
- **יופי** (красота): עדין, אלגנטי, מרשים, חם, עמוק
- **נוחות** (комфорт): רגוע, ללא כאב, עדין, נעים

### Сенсорные слова:

- **Цвет**: חלבי, קרמל, ורוד מאובק, בורדו, שוקולד, טרקוטה

# РАЗДЕЛ 6: АНТИПАТТЕРНЫ (примеры плохого текста + исправление)

## АНТИПАТТЕРН 1: Пустые слова

❌ **Плохо:**
```
Красивый и качественный маникюр в лучших традициях! 
Профессиональный подход к каждой клиентке! 
Запишись и убедись сама! 💅💅💅
```

✅ **Хорошо:**
```
Нюдовый камуфляж на миндале 💅

Трёхслойное нанесение, запечатывание торца, аккуратная кутикула.
```

❌ **Плохо:**

✅ **Хорошо:**

## АНТИПАТТЕРН 4: Скопированный текст / однообразие

❌ **Плохо:**
```

Пост 2: «Стойкое покрытие до 4 недель! Свободные окна!»
Пост 3: «Стойкое покрытие до 4 недель! Ваш маникюр ждёт!»
```

✅ **Хорошо:**
```
Пост 1: «Через 3 недели. Ни одного скола» (результат)
Пост 2: «Почему у одних маникюр живёт месяц, а у других неделю?» (образование)
```

Все три раскрывают USP стойкости — но каждый через ДРУГОЙ угол.

# РАЗДЕЛ 7: РЕКЛАМНЫЙ КОПИРАЙТИНГ (Meta Ads)

## 7.1 Правила рекламного текста

### Ключевые отличия от organic поста:

1. **Первые 125 символов = всё.** Meta показывает только начало текста. Если человек не нажмёт «ещё» — он видит ТОЛЬКО hook. Самое важное — в первые 2 строки.

2. **CTA в тексте необязателен.** Meta добавляет кнопку автоматически (Send WhatsApp Message / Отправить сообщение). Текст может заканчиваться оффером, не дублируя CTA кнопки.

3. **Короче = лучше для beauty.** Рекомендуемая длина primary text — до 125 символов для максимального видимого текста. Но можно до 3-5 строк, если hook сильный.

4. **Meta рекомендует до 5 вариантов primary text.** Давать 3 варианта разной длины: короткий (1-2 строки), средний (3-4 строки), длинный (5-6 строк). Meta сама протестирует.

5. **Headline — 3-5 слов.** «Маникюр в [городе] | Запись открыта» — достаточно. Не пытайся впихнуть всё.

6. **Цена в рекламе — если конкурентная.** 120₪ за маникюр — рыночная цена, можно показывать. Если выше рынка — лучше не показывать, привлекать ценностью.

## 7.3 DO и DON'T для рекламы

### DO:
- Давай 3 варианта длины — пусть Meta тестирует

- Headline: коротко, с городом, с типом действия

### DON'T:
- Не дублируй CTA кнопки текстом (Meta добавит сама)
- Не пиши текст длиннее 5-6 строк для первого теста
- Не используй более 2 эмодзи в рекламе
- Не обещай невозможного («100% гарантия»)
- Не ставь цену если она выше рынка

# РАЗДЕЛ 10: REWRITE ГАЙД (улучшение текста после Scorer)

## 10.1 Когда Scorer возвращает текст

Scorer может вернуть текст по следующим причинам:
- **weak_hook** — слабый hook, не цепляет
- **too_generic** — слишком общий текст, нет конкретики
- **too_long** — текст длиннее нормы для типа поста
- **repetitive** — hook или USP повторяет предыдущие посты
- **wrong_tone** — тон не соответствует бренду
- **no_cta** — отсутствует CTA
- **cliche** — использованы стоп-слова или клише
- **fake_feeling** — текст звучит неискренне

## 10.2 Техники исправления

### Если weak_hook:
1. Попробуй ДРУГУЮ формулу hook (если был вопрос — замени на мини-историю)
2. Добавь конкретику (число, срок, деталь)
3. Сделай короче — hook должен быть 1-2 строки, не больше
4. Тест: прочитай hook вслух. Ты бы остановился? Если нет — переписывай.

**Было:** «Красивый маникюр для вас ✨»
**Стало:** «26 дней. Ни одного скола. Листай — покажу 👇»

### Если too_generic:
1. Замени прилагательные на факты: «красивый» → «нюдовый на миндале»
2. Добавь числа: «долго держится» → «до 4 недель без сколов»
3. Добавь сенсорное описание: «хороший цвет» → «как латте с корицей — тёплый и сдержанный»

**Было:** «Качественный маникюр от профессионального мастера»
**Стало:** «Камуфляж Kodi #32 на мягком миндале. 3 слоя + запечатывание торца. Результат — на фото через 20 дней»

### Если too_long:
1. Убери повторения (одна и та же мысль разными словами)
2. Убери «воду» (вводные фразы, слова-связки без смысла)
3. Сократи до формулы: hook + 2-3 абзаца + CTA
4. Правило: если предложение можно убрать и смысл не меняется — убирай

### Если repetitive:
1. Проверь список предыдущих hooks из brief'а
2. Используй ДРУГУЮ формулу hook (всего 8 формул — всегда есть альтернатива)
3. Найди новый угол для той же темы: если про стойкость — через историю клиентки, через процесс, через сравнение, через миф

### Если wrong_tone:
- **Слишком формально:** убери «вы можете», «наша студия предлагает». Пиши проще.
- **Слишком фамильярно:** убери «девочки!!!», «ну вот вам», «реально крутой». Добавь уверенности.
- **Слишком продающе:** убери «только сегодня», «не упустите», «уникальное предложение». Покажи ценность.

### Если cliche:
1. Найди стоп-слово → замени по W-BAN-001 / W-BAN-002
2. Если клише в hook — полностью перепиши первую строку
3. Если клише в body — замени конкретикой

### Если fake_feeling:
1. Убери восторженные оценки («невероятный!», «потрясающий!»)
2. Добавь деталь, которая создаёт правдоподобность (конкретный день, ситуация)
3. Снизь пафос — тёплый тон, а не восхищённый

## 10.3 Чеклист самопроверки перед отправкой

- [ ] Hook уникален (нет в списке предыдущих hooks)?
- [ ] Первые 125 символов — цепляют?
- [ ] Нет стоп-слов из W-BAN-001 / W-BAN-002?
- [ ] Есть конкретика (числа, названия, описания)?
- [ ] Тон = «тёплый профессионал»?
- [ ] CTA присутствует и уместен?
- [ ] Длина в рамках нормы для типа поста?
- [ ] USP не повторяется дословно?
- [ ] Текст на ОДНОМ языке (не смешан)?
- [ ] Эмодзи: 2-5, уместные?
- [ ] Прочитал вслух — звучит естественно?
- [ ] Хештегов не более 5, все релевантны?

| Параметр | Русский | Иврит |
|---|---|---|
| Формальность | Средняя (на «вы» в feed) | Низкая (на «את», неформально) |
| Длина текста | Можно длиннее, ЦА читает | Короче, прямее, по делу |
| Эмоциональность | Больше (истории, чувства) | Меньше (факты, результат) |
| Юмор | Лёгкий, самоирония | Более прямой, неформальный |
| CTA | Мягкий → прямой | Сразу прямой |

## 11.3 Тест на правильный тон

Прочитай текст вслух и ответь:
1. Звучит как живой человек или как робот / реклама?
2. Ты бы сказал это клиентке лично?
3. Тебе бы хотелось дочитать до конца?
4. После прочтения — понятно что делать дальше?

Если хоть один ответ «нет» — переписывай.
